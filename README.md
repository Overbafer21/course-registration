# Онлайн-система регистрации на курсы

Небольшое desktop-приложение на Java Swing для регистрации студентов на курсы.

## Возможности

- Добавление студента с полями:
  - имя,
  - ID,
  - название курса,
  - email.
- Отображение всех регистраций в таблице.
- Кнопка очистки формы.
- Предустановленные тестовые записи при запуске.

## Стек

- Java (Swing, AWT)

## Как запустить

1. Убедитесь, что установлен JDK 11+.
2. Скомпилируйте приложение:

```bash
javac *.java
```

3. Запустите:

```bash
java Main
# или
java CourseRegistrationSystem
```

## Структура

- `CourseRegistrationSystem.java` — основной код интерфейса и логики регистрации.
